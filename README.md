# BASORIA - Premium Indonesian Meatball Restaurant

A premium restaurant website for BASORIA, featuring authentic Indonesian bakso (meatball) cuisine.

## Project Structure

```
/
├── index.html          # Home page
├── menu.html           # Menu page with filtering
├── about.html          # About page with brand story
├── gallery.html        # Photo gallery with filters
├── contact.html       # Contact form and locations
├── css/
│   └── style.css       # Main stylesheet
├── js/
│   └── script.js       # Main JavaScript file
├── package.json        # Vite configuration
├── vite.config.js      # Vite config
└── README.md
```

## Features

### Pages
- **Home**: Hero section, about preview, featured menu, statistics, testimonials
- **Menu**: Full menu with search and category filters
- **About**: Brand story, vision/mission, pillars, team
- **Gallery**: Photo gallery with hover animations and lightbox
- **Contact**: Contact form, Google Maps, branch locations

### Design
- **Color Palette**:
  - Primary: #6B0F1A (Deep Maroon)
  - Secondary: #0D1B2A (Dark Navy)
  - Accent: #C9A84C (Gold)
  - Background: #FFFFFF

- **Typography**:
  - Headings: Cormorant Garamond (serif)
  - Body: Inter (sans-serif)

### Features
- Responsive design (desktop, tablet, mobile)
- Smooth scroll animations
- Fade-in animations on scroll
- Interactive menu cards with hover effects
- Category filtering for menu and gallery
- WhatsApp integration for orders
- Floating WhatsApp button
- Sticky navigation with scroll effect

## Tech Stack

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Styling**: Custom CSS + Bootstrap 5
- **Icons**: Bootstrap Icons
- **Fonts**: Google Fonts (Cormorant Garamond, Inter)
- **Images**: Pexels (stock photos)
- **Build Tool**: Vite

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```
